#Project 5:  Neighborhood Map

##About
Application shows parks and their locations in Middletown, NJ area. 
On click marker pull image from Flickr API and display them in popup window.

##Frameworks and APIs used:
- jQuery
- Knockout.js
- Google Maps
- Flickr
